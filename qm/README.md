# qm

A mailbox interface for the terminal, built on the Quarb engine.

`qm` browses and queries mail — Maildir, mbox, and IMAP — using
[Quarb](https://quarb.org), the query language for *arbors*
(tree-spanned graphs). Mailboxes, folders, threads, and messages
are one queryable arbor.

## Install

```sh
cargo install qm
```

See [quarb.org](https://quarb.org) for the language and the email
cookbook.
